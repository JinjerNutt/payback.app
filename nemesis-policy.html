Nemesis DCA Trade Planner

Privacy Policy

Last updated: April 2026

This Privacy Policy applies to the Nemesis – DCA Trade Planner mobile application ("Application"), developed by Joshua Crellin ("Service Provider") and provided as a paid application. The Application is provided "AS IS".

Data Collection and Use

The Application does not require users to create accounts or provide any personally identifiable information to function.

All data entered into the Application — including exchange API keys, trade settings, strategy configurations, journal entries, and watchlist data — is stored locally on your device only, using standard iOS secure storage. No personal data, API credentials, or trading information is uploaded, synced, or shared with the Service Provider or any external servers.

If you delete the Application, all locally stored data is permanently removed.

Exchange API Keys

Nemesis supports connecting to multiple cryptocurrency exchanges, including Bybit, Binance, BingX, Bitget, Blofin, Gate.io, Kraken, KuCoin, MEXC, and OKX. Any API key and secret you provide for a supported exchange is stored exclusively on your device using iOS secure storage.

When you choose to import your balance or trade history, your credentials are transmitted directly from your device to the relevant exchange for the sole purpose of fetching that data. The Service Provider never receives, stores, or has access to your API credentials.

API keys should be configured with read-only permissions only. You should never enable withdrawal permissions on any API key used with this Application.

Live Price Data

The Application fetches live cryptocurrency prices from Bybit's public market API to power the watchlist and price alert features. These requests do not include any personal information, account credentials, or identifiable device data. No user data is sent as part of price lookups.

Push Notifications

The Application may request permission to send push notifications in order to deliver price alerts when a watched asset reaches your target price. Notifications are generated entirely on-device by iOS and are not routed through any external server. If you do not grant notification permission, price alerts will not be delivered but the rest of the Application will continue to function normally.

You can manage or revoke notification permissions at any time in iOS Settings → Notifications → Nemesis.

Face ID / Biometric Lock

The Application offers an optional Face ID lock to protect access to your exchange API keys and app data. Biometric authentication is handled entirely by iOS using Apple's LocalAuthentication framework. The Service Provider does not collect, store, or have access to any biometric data.

Trading Journal

Journal entries — including trade notes, emotion tags, and performance statistics — are stored locally on your device by default. The Application includes an optional iCloud sync feature, which, if enabled, stores your journal data in your personal iCloud Key-Value store using Apple's NSUbiquitousKeyValueStore. This data is associated with your Apple ID and is governed by Apple's iCloud privacy policy. The Service Provider does not have access to this data. iCloud sync can be enabled or disabled at any time in the Application's settings.

Third-Party Services

The Application communicates with the following external services:

  • Cryptocurrency exchanges (Bybit, Binance, BingX, Bitget, Blofin, Gate.io, Kraken, KuCoin, MEXC, OKX) — only when you explicitly connect an account or trigger a balance/history import. Communications are governed by the privacy policy of the respective exchange.
  • Bybit public market API — for live price data. No personal information is sent.
  • Apple App Store / StoreKit — for processing in-app subscription purchases. Apple's privacy policy applies to all purchase transactions.

The Application does not integrate any advertising networks, analytics services, or other third-party data processors.

Financial Risk Disclaimer

Nemesis is a calculator and trade planning tool. It is not a financial advisor, automated trading bot, or investment service. All trading decisions are made by the user. Cryptocurrency trading involves significant risk of loss. The Service Provider accepts no liability for trading losses incurred through use of the Application.

Children's Privacy

The Application is intended for adults who engage in cryptocurrency trading. The Service Provider does not knowingly collect any personally identifiable information from children under the age of 13.

Security

The Service Provider takes reasonable measures to safeguard the information processed by the Application. Because all user data remains on the device, data exposure risks are minimized. Users are responsible for maintaining the security of their own device and exchange API credentials.
