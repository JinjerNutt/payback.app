Privacy Policy

Last updated: January 2026

This Privacy Policy applies to the Payback – Debt Tracker mobile application ("Application"), developed by Joshua Crellin ("Service Provider") and provided as a Freemium service. The Application is provided “AS IS”.

Data Collection and Use

The Application does not require users to create accounts or provide any personally identifiable information to function.

All financial data entered into the Application is stored locally on your device only, using standard iOS secure storage. No personal data is uploaded, synced, or shared with the Service Provider or any external servers.

If you delete the Application, all locally stored data is permanently removed.

Children’s Privacy

The Application is suitable for general audiences and may be used by children. The Service Provider does not knowingly collect any personally identifiable information from children under the age of 13. The Application does not include user accounts, social features, or data submission mechanisms that would allow the collection of personal information from children.

Security

The Service Provider takes reasonable measures to safeguard the information processed by the Application using appropriate physical, electronic, and procedural protections. Because all user data remains on the device, data exposure risks are minimized.

Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page by updating the “Last updated” date. Continued use of the Application constitutes acceptance of the updated policy.

Consent

By using the Application, you consent to this Privacy Policy.
